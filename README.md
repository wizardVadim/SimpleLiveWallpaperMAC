# Simple Live Wallpaper for Mac

## EN

A free application to use MP4 files as live wallpapers on macOS.

### Changelog

- **v1.1.0**
  | Функция |
  |---------|
  | You can use more than one screen |
  | Wallpapers will be starting after you start the app |

- **v1.0.0**
  | Feature |
  |---------|
  | Ability to upload wallpapers from your MacBook |
  | From the list of uploaded wallpapers, you can choose which ones you want to play now |
  | Start / Stop button |
  | Ability to remove wallpapers from the available list |

## RU

Бесплатное приложение для установки живых обоев на систему macOS.

### Изменения

- **v1.1.0**
  | Функция |
  |---------|
  | Добавлена поддержка нескольких экранов |
  | При включении будет воспроизведена очередь обоев (сохранение очереди) |

- **v1.0.0**
  | Функция |
  |---------|
  | Возможность загрузить обои с вашего MacBook |
  | Из списка загруженных обоев можно выбрать те, которые вы хотите воспроизвести сейчас |
  | Кнопка Старт / Стоп |
  | Возможность удалить из списка доступных |
